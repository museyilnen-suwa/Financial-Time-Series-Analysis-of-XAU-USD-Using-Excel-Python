# XAU/USD Financial Market Data Analysis

## Overview

This project presents an exploratory data analysis of historical XAU/USD (Gold vs US Dollar) market data collected at 15-minute intervals.

The objective was to demonstrate practical data analysis skills by cleaning, validating, transforming, and visualizing a large financial dataset containing over 490,000 observations.

---

## Objectives

- Clean raw market data
- Detect duplicates and missing values
- Transform timestamps into analytical features
- Calculate price movement metrics
- Analyze long-term trends
- Build an Excel dashboard
- Produce statistical insights
